# MouseHook - Multiple screen mouse helper #

## About ##

MouseHook ver 0.1, Freeware edition

## How to build ##

To build *MouseHook* you need [Flat assembler (FASM)](http://flatassembler.net/).  
Just run this commands:

	fasm hook_dll.asm
	fasm MouseHook.asm
